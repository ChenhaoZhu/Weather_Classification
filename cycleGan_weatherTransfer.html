class RegressionCNN(nn.Module):
    def __init__(self, kernel, num_filters):
        # first call parent's initialization function
        super().__init__()
        padding = kernel // 2

        self.downconv1 = nn.Sequential(
            nn.Conv2d(3, num_filters, kernel_size=kernel, padding=padding),
            nn.BatchNorm2d(num_filters),
            nn.ReLU(),
            nn.MaxPool2d(2),)
        self.downconv2 = nn.Sequential(
            nn.Conv2d(num_filters, num_filters*2, kernel_size=kernel, padding=padding),
            nn.BatchNorm2d(num_filters*2),
            nn.ReLU(),
            nn.MaxPool2d(2),)

        self.rfconv = nn.Sequential(
            nn.Conv2d(num_filters*2, num_filters*2, kernel_size=kernel, padding=padding),
            nn.BatchNorm2d(num_filters*2),
            nn.ReLU())

        self.upconv1 = nn.Sequential(
            nn.Conv2d(num_filters*2, num_filters, kernel_size=kernel, padding=padding),
            nn.BatchNorm2d(num_filters),
            nn.ReLU(),
            nn.Upsample(scale_factor=2),)
        self.upconv2 = nn.Sequential(
            nn.Conv2d(num_filters, 3, kernel_size=kernel, padding=padding),
            nn.BatchNorm2d(3),
            nn.ReLU(),
            nn.Upsample(scale_factor=2),)
        self.finalconv = nn.Conv2d(3, 3, kernel_size=kernel, padding=padding)

    def forward(self, x):
        out = self.downconv1(x)
        out = self.downconv2(out)
        out = self.rfconv(out)
        out = self.upconv1(out)
        out = self.upconv2(out)
        out = self.finalconv(out)
        return out

def get_torch_vars(xs, ys, gpu=False):
    """
    Helper function to convert numpy arrays to pytorch tensors.
    If GPU is used, move the tensors to GPU.

    Args:
      xs (float numpy tenosor): greyscale input
      ys (int numpy tenosor): rgb as labels
      gpu (bool): whether to move pytorch tensor to GPU
    Returns:
      Variable(xs), Variable(ys)
    """
    xs = torch.from_numpy(xs).float()
    ys = torch.from_numpy(ys).float()
    if gpu:
        xs = xs.cuda()
        ys = ys.cuda()
    return Variable(xs), Variable(ys)

Define the Generator and Discriminator for weather changing.¶

  • Cyclegan has two generators, now we just make these two generators the same structure, which make use of skip connection to avoid gradient vanishing.
  • For discriminator, we just use fully connected layers with appropriate drop-out.

            nn.BatchNorm2d(num_colours),
            nn.ReLU(),
            nn.Upsample(scale_factor=2),)      
        self.finalconv = nn.Conv2d(num_in_channels + num_colours, num_colours, kernel_size=kernel, padding=output_padding)
        

    def forward(self, x):
        ############### YOUR CODE GOES HERE ###############
        ###################################################
        out1 = self.downconv1(x)
        out2 = self.downconv2(out1)
        out3 = self.rfconv(out2)

        input4 = torch.cat((out2, out3), 1)
        out4 = self.upconv1(input4)

        input5 = torch.cat((out1, out4), 1)
        out5 = self.upconv2(input5)

        input6 = torch.cat((x, out5), 1)
        out = self.finalconv(input6)

        return out

Run the Train codes to finalize the model¶

  • For cyclegan algorithm, the most important part is the loss function because it contains two generators and potentially more than 1 discriminator.
  • In the training part, we first train the first generator to make rainy image to cloudy image, and then transform de-rainy image to rainy image; Second, we use the second generator to make cloudy image to rainy image, and then transform rainy image to cloudy image. In such a loop, we simply implement the idea of cyclegan.
  • For our cyclegan model. We do not get a satisfied version because it do not change the origiinal image very well. Thus we think that it may because that the process in train function has some mis-understanding. We will spend time on that in the future.
